# House-Sales-in-King-Country-USA

## Project Overiview
This project simulates the role of a Data Analyst working for a Real Estate Investment Trust (REIT) that plans to invest in residential properties. The objective is to analyze and predict housing prices based on various property features.

## Goal 
Build a predictive model to estimate the market price of a house using attributes such as:
- Square footage
- Number of bedrooms
- Number of floors
- And other relevant features
